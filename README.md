# TypeRush - Typing Speed Game

TypeRush is a front-end typing speed test application built using HTML, CSS, and JavaScript.  
It allows users to measure typing speed (WPM), accuracy, and completion time across multiple difficulty levels.

This project was developed as a school assignment to demonstrate DOM manipulation, event handling, responsive design, and browser-based data storage using LocalStorage. No backend or external libraries are used.

---

## Live Demo

https://typerushwafi.netlify.app/

---

## Features

- Three difficulty levels: easy, medium, hard  
- Real-time typing validation (character by character)  
- Words per minute (WPM) calculation  
- Accuracy calculation based on correct characters  
- Timer tracking total completion time  
- Progress tracking during typing  
- Performance evaluation system based on speed  
- Score storage using LocalStorage per difficulty  
- Leaderboard displaying best scores  
- Restart and reset functionality without page reload  
- Responsive design for desktop and mobile devices

## Prompt History

This project was developed iteratively using AI assistance. Key prompts included:

- Create a typing speed game using HTML, CSS, and JavaScript only  
- Implement WPM, accuracy, and timer tracking  
- Add multiple difficulty levels with different texts  
- Store and retrieve scores using LocalStorage  
- Make the interface responsive for mobile and desktop  
- Fix GitHub deployment and push issues  
- Debug scoring and game completion logic

## Technologies Used

- HTML5 for structure  
- CSS3 for styling and layout  
- JavaScript (Vanilla) for game logic and interactivity  
- LocalStorage API for saving user scores in the browser
